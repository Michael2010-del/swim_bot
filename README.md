


# 🏊‍♂️ Swimming Bot — Telegram-бот о плавании

Интерактивный Telegram-бот, который помогает изучать техники плавания, даёт полезные советы и сравнивает разные стили.

## 📋 Возможности

- **Информация о 4 стилях плавания**:
  - Кроль
  - Брасс
  - Баттерфляй
  - Плавание на спине

- **Совет дня** — случайные полезные советы для пловцов

- **Сравнение стилей** — скорость и особенности каждого стиля

- **Удобное меню** с кнопками для навигации

## 🚀 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и начало работы |
| `/menu` | Показать главное меню со стилями |
| `/help` | Список доступных команд |
| `/info` | Информация о боте |

## 🎯 Как использовать

1. Запустите бота командой `/start`
2. Откройте меню через `/menu`
3. Выберите стиль плавания для получения информации
4. Используйте кнопки:
   - **Совет дня** — получите случайный совет
   - **Сравнить стили** — узнайте особенности каждого стиля

## 🛠️ Технологии

- Python 3.7+
- pyTelegramBotAPI (Telebot)
- Random (стандартная библиотека)

## 📦 Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/your-username/swimming-bot.git
cd swimming-bot
```

### 2. Установите зависимости

```bash
pip install pyTelegramBotAPI
```

### 3. Настройте токен бота

Создайте файл `config.py` и добавьте в него токен вашего бота:

```python
TOKEN = 'ваш_токен_от_BotFather'
```

### 4. Запустите бота

```bash
python bot.py
```
